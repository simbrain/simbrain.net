Simbrain 3.05

Simbrain 3.05 is the first version of Simbrain packaged as a normal desktop application. Note that you may have to use right-click to open (Mac) or click show details and run anyway (PC). We are working to resolve these issues now. If other problems please arise let us know here. If needed you can run the earler 3.04 version, but note that you should run it using java 8.
