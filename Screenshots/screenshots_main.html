The SOM Smells workspace. Here to demonstrate that we support self organizing maps (having this alongside spiking computational neuroscience models really shows our flexibility), as well as give a look at one of the more common worlds: Odor World.

 
This is just the cerebellum script available in the script menu. Chosen to demonstrate the flexibility of Simbrain’s nodes. They don’t have to be used explicitly as neurons and instead can be used to represent whole brain areas or the activity of a cluster of neurons for the purposes of more coarse grained simulations and models.

 
This is an instantiation of a SORN (Self organizing recurrent network), see Lazar, Pipa, & Trisect 2009. This is here as an example of several things: 1. Scripting & Flexibility 2. Large scale (the network has something like 5000 neurons and 3.7 Million synapses) 3. The ability to load a .csv file as inputs to the network and simply run it (so long as the input neuron group is in input mode). 4. Interesting network structure.

 

More examples of visualizers. This time the emphasis (especially alongside the giant SORN workspace) is that Simbrain can be used to analyze small scale interactions as well as run giant simulations. It can be used as a teaching tool to get across the concept of STDP as easily as it can be used to educate students as to the dynamics of a reservoir.

 

I like this one because it displays another one of the worlds as well as shows a trained Simple recurrent network (yet another model we support). The PCA plot to the left I think is extra impressive though. I coupled it to the hidden layer and low and behold it looks like we have 5 or 6 distinct clusters. When the network is run you can actually see how certain strings of letters move the hidden layer into or from one of those clusters. The large one at the top appeared to be where the network went when there were a lot of us in the input history.

 

This neural network’s synapse weights were extrapolated from a transfer entropy matrix which was taken from a dissociated culture of living neural tissue. Each node is placed at the coordinates where a 512 electrode array was used to ascertain the location of individual neurons. Synapses are where TE-Analysis indicates real synaptic connections are the most likely to exist. The neuron model used here is the adaptive exponential integrate and fire model. Again this shows our flexibility. You can load a weight matrix taken from real neurons as well as their positions and viola, you can put that model in Simbrain.
